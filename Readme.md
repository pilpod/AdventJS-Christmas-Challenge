# AdventJS

Advent JS challenge

## Pre-requirement

- node.js & npm

## Dependencies

- jest
- @babel/core
- @babel/preset-env

## Installation

```
$ npm install
```


## Run tests

```
$ npx jest
or
$ npm run test
```
You can filter test with ...
```
$ npx jest -t '<test-name>'
or
$ npm run filter '<test-name>'
```
